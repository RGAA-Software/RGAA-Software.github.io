#### 1. If you cannot see the screen after connecting
##### 1.1 Try waiting a few seconds, the screen may be static and not triggering capture
##### 1.2 If you see a black screen, try triggering Ctrl+Alt+Delete, or Refresh Screen, or Restart Render
![](assets/qa_1.png)

##### 1.3 If there are rendering issues, screen artifacts, try forcing software decoding in settings (Force Software Decode), if the remote machine is old, you can force GDI capture (Force GDI capture), if your client machine is old, you can disable Vulkan rendering (Disable Vulkan render)
![](assets/qa_2.png)
![](assets/qa_3.png)
